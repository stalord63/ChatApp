# ReaTime ChatApp
> REAL TIME CHAT APPLICATION SERVICE USING SOCKETS.

### Installation
----

It requires [Node.js](https://nodejs.org/) v4+ to run.

----

### Install the dependencies and devDependencies and **start the server**.

```sh
$ cd ChatApp
$ npm install express
$ npm link express
$ node index.js


Run https://localhost:3000
```
